## GLTerm for VGA32
A fork of the FabGL ANSI-Terminal with parts of the VGA32 RunCPM-port<br>
useable as serial TTL-Terminal (GP(IO) 20&21) for the Picomite<br>
(Raspberry Pi Pico with MMBASIC)<br>
